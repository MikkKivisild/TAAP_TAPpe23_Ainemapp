1. Mis on Flowchart diagrammid
    Flowchart on pilt protsessi eraldi etappidest järjestikuses järjekorras. See on
    üldine tööriist, mida saab kohandada väga erinevatel eesmärkidel ja mida saab kasutada
    erinevate protsesside, näiteks tootmisprotsessi, haldus- või teenindusprotsessi või
    projektiplaani kirjeldamiseks.
  2. Kus neid kasutatakse
    Neid kasutatakse laialdaselt mitmes valdkonnas sageli keeruliste protsesside
    dokumenteerimiseks, uurimiseks, planeerimiseks, täiustamiseks ja edastamiseks
    selgete ja hõlpsasti mõistetavate diagrammide kujul.
  3. eri kujundite tähendused
    Enim kasutatud kujundid
    Process symbol - Tuntud ka kui "Action symbol", see kujund tähistab protsessi,
    tegevust või funktsiooni. See on vooskeemides kõige laialdasemalt kasutatav sümbol.

    Start/End symbol - Tuntud ka kui "Terminator Symbol", see sümbol tähistab tee
    alguspunkte, lõpp-punkte ja võimalikke tulemusi. Sageli sisaldab kujund "Start" või "End".

    Document symbol - Represents the input or output of a document, specifically.
    Examples of and input are receiving a report, email, or order. Examples of an output
    using a document symbol include generating a presentation, memo, or letter.

    Decision symbol - Tähistab küsimust, millele tuleb vastata — tavaliselt jah/ei või
    õige/vale. Vooskeemi tee võib seejärel jaguneda erinevateks harudeks sõltuvalt
    vastusest või järgnevatest tagajärgedest.

    Connector symbol - Usually used within more complex charts, this symbol connects
    separate elements across one page.

    Off-Page Connector/Link symbol - Frequently used within complex charts, this symbol
    connects separate elements across multiple pages with the page number usually placed
    on or within the shape for easy reference.

    Input/Output symbol - Also referred to as the “Data Symbol,” this shape represents data
    that is available for input or output as well as representing resources used or generated.
    While the paper tape symbol also represents input/output, it is outdated and no longer in
    common use for flowchart diagramming.

    Comment/Note symbol - Kontekstiga koos paigutatuna lisab see sümbol vajalikke selgitusi või
    kommentaare määratud vahemikus. Selle võib katkendjoonega ühendada ka vooskeemi vastava osaga.
